## 關鍵按鈕位置

1. **進入管理後台**：主畫面右下角 → ⚙️ 設定圖示
2. **上傳檔案**：左側面板 → 📎 選擇檔案按鈕
3. **下載範本**：左側面板 → 📥 下載 Excel 範本
4. **新增條目**：左側面板 → ➕ 手動新增條目
5. **儲存知識庫**：右上角 → 💾 儲存知識庫（綠色按鈕）
6. **刪除條目**：表格最右側 → 🗑️ 刪除圖示
7. **返回主頁**：右上角 → ✕ 返回按鈕
8. **重新載入**：主畫面右下角 → 🔄 重新載入圖示

## 顏色說明

- 🟣 紫色漸層：主視覺背景
- 🟢 綠色：儲存、成功操作
- 🔴 紅色：刪除、高風險
- 🔵 藍色：主要操作按鈕
- ⚪ 白色：內容區塊
- 🟡 黃色：警告訊息

## 圖示說明

- 🗄️ 資料庫：知識庫
- 📤 上傳：匯入檔案
- ✏️ 編輯：編輯模式
- 💾 磁碟：儲存
- 🗑️ 垃圾桶：刪除
- ⚙️ 齒輪：設定/管理
- 🔄 循環：重新載入
- 📥 下載：下載範本
- ➕ 加號：新增
- ✕ 叉號：關閉/返回
- ✅ 勾號：成功
- ❌ 叉號：失敗/禁止
- 🔐 鎖頭：密碼保護
